# SmartBiz — Modern Websites for Kenyan Businesses

SmartBiz is a web development agency focused on building fast, modern, and mobile-first websites for businesses in Kenya.

## Features
- **Fast Delivery**: Websites ready in 1–2 days.
- **Mobile Friendly**: Optimized for all screen sizes.
- **Conversion Focused**: Designed to turn visitors into customers.
- **WhatsApp Integration**: Direct call-to-action for immediate engagement.

## Services & Pricing
- **Starter**: KES 3,000
- **Business**: KES 7,000
- **Premium**: KES 15,000

- CSS3 (Custom Styles)
- JavaScript (Scroll Animations)
